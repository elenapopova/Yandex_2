## Для запуска модуля используется файл index.js.

## Последовательность действий файла index.js:
1. Подключение модуля.
2. Чтение данных из файла data.json в массив data.
3. Возвращение результата в объект newData.
4. Вывод результата на экран.

## Последовательность действий модуля obfuscator.js:
1. Удаление повторяющихся старых имен классов.
2. Подсчет вхождений уникальных имен в исходном массиве.
3. Сортировка массива single.
4. Составление нового имени класса.
